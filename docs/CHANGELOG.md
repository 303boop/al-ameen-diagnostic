# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-02

### Added
- Initial release
- Complete appointment booking system
- Patient dashboard with appointment history and reports
- Lab dashboard with booking search and report upload
- Admin dashboard with complete management features
- Bilingual support (English & Bengali)
- Dark/Light theme toggle
- Guest booking without login
- Serial number system starting from 1
- Unique booking ID format (ALM-DDMMYY-XXX)
- Coupon code system
- Role-based access control (Admin, Lab, Patient)
- Responsive design for all devices
- Modern teal medical theme
- Smooth animations and transitions
- Image galleries with Swiper
- Doctor and test management
- Audit logging system
- In-app notifications
- Report upload and download
- Clinic settings management

### Security
- Row Level Security (RLS) on all tables
- Secure authentication with Supabase
- Password hashing
- Protected routes based on roles
- Private file storage for reports

### Documentation
- Complete README with setup instructions
- API documentation
- Database schema documentation
- Setup guide
